# Desert Fern

**WORK IN PROGRESS**

## Gameplay

You are a fern searching for your cactus friend. Traverse the desert to save her.

### Controls

w - jump  
a - go backward  
s - duck  
d - go forward  
shift - run

Note that duck prevents using the other keys.
